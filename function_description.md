# 함수 설계

## main()

### 함수명

main()

### 정의

menu 화면을 호출하고 사용자로부터 메뉴를 입력받아 해당 menu 를 실행한다.

### 반환

x

### 입력

x

### 처리과정

```
1. menu 가 0~5 일 동안 반복한다.
2. menu 화면을 출력한다.
3. menu 를 입력 받는다.
4. 해당 menu 맞는 함수를 호출한다.
```

## 주소정보를 입력한다

### 함수명

inputAddressInfo()

### 정의

주소정보를 입력받아 인자로 전달받은 주소록에 추가한다.

### 반환

addressBook : addressInfo(10) : 주소록

### 입력

addressBook : addressInfo(10) : 주소록

### 처리과정

```
1. 주소정보 개수가 가득 찼는지 검사한다.
	1.1. 가득 : 에러 메시지를 출력하고 주소록을 반환한다.
2. 이름을 입력 받는다.
3. 전화번호를 입력 받는다.
4. 주소를 입력 받는다.
5. 입력된 주소정보를 출력한다.
6. 추가 여부를 입력 받는다.
	6.1. no : 메시지 출력하고 주소록을 반환한다.
7. 비어있는 주소록에 주소정보를 추가한다.
8. 주소록을 반환한다.
```

## 주소정보를 삭제한다

### 함수명

removePersonalInfo()

### 정의

삭제할 이름을 입력 받아 주소록에서 해당 주소정보를 삭제처리 한다.

### 반환

addressBook : personalInfo(10) : 주소록

### 입력

addressBook : personalInfo(10) : 주소록

### 처리과정

```
1. 주소정보 개수가 비어 있는지 검사한다.
	1.1. 빔 : 에러 메시지를 출력하고 주소록을 반환한다.
2. 이름을 입력 받는다.
3. 주소록에서 해당 이름으로 검색한다.
4. 검색 결과가 비어 있는지 검사한다.
	4.1. 빔 : 에러 메시지를 출력하고 주소록을 반환한다.
5. 검색 결과가 2개 이상인지 검사한다.
	5.1. 2개 이상 : 중복 이름의 주소정보를 출력한다.
	5.2. 2개 이상 : 삭제한 주소정보 번호를 입력 받는다.
6. 삭제여부를 입력 받는다.
	6.1. no : 메시지를 출력하고 주소록을 반환한다.
7. 삭제할 주소정보를 삭제처리 한다.
8. 주소록을 반환한다.
```

## 주소정보를 수정한다

### 함수명

modifyPersonalInfo()

### 정의

수정할 이름을 입력 받아 주소록에서 해당 주소정보를 변경 한다.

### 반환

addressBook : personalInfo(10) : 주소록

### 입력

addressBook : personalInfo(10) : 주소록

### 처리과정

```
1. 주소정보 개수가 비어 있거나 가득 찼는지 검사한다.
	1.1. yes : 에러 메시지를 출력하고 주소록을 반환한다.
2. 이름을 입력 받는다.
3. 주소록에서 해당 이름으로 검색한다.
4. 검색 결과가 비어 있는지 검사한다.
	4.1. 빔 : 에러 메시지를 출력하고 주소록을 반환한다.
5. 검색 결과가 2개 이상인지 검사한다.
	5.1. 2개 이상 : 중복 이름의 주소정보를 출력한다.
	5.2. 2개 이상 : 수정할 주소정보 번호를 입력 받는다.
6. 수정여부를 입력 받는다.
	6.1. no : 메시지를 출력하고 주소록을 반환한다.
7. 이름을 입력 받는다.
8. 전화번호를 입력 받는다.
9. 주소를 입력 받는다.
10. 입력된 주소정보를 해당 주소정보에 덮어쓴다.
11. 변경된 주소정보를 출력한다.
12. 주소록을 반환한다.
```

## 주소록이 가득 찼는지 검사한다.

### 함수명

isFullAddressBook()

### 정의

주소록이 가득 찼는지 검사하여 결과를 반환한다.

### 반환

isFull : boolean : 가득찼는지 여부 ( true - 가득, false - 여부 있음 )

### 입력

addressBook : personalInfo(10) : 주소록

### 처리과정

```
1. 주소록을 순회한다.
	1.1. 주소정보의 삭제여부 속성이 True 인지 확인한다.
		1.1.1. True : 삭제여부결과에 True 를 설정한다
		1.1.2. 순회를 종료한다
2. 삭제여부결과가 참인지 검사한다
	2.1. True(여부있음): False 를 반환한다
	2.2. False(가득): True 를 반환한다
```

## 주소록에서 이름으로 검색하여 결과 주소정보 배열을 반환한다.

### 함수명

_searchPersonalInfo()

### 정의

주소록에서 인자로 들어온 이름을 검색하여 검색된 주소정보를 결과 배열에 채워 반환한다.

### 반환

searchedArrary : personalInfo(10) : 검색된 주소 배열

### 입력

name : char(7) : 검색할 이름
addressBook : personalInfo(10) : 주소록

### 처리과정

```
1. 검색 주소 배열을 초기화한다.
2. 주소록을 순회한다.
	2.1. 주소정보의 삭제여부 속성이 False 인지 확인한다.
		2.1.1. yes : 해당 주소정보의 이름과 입력 이름이 같은지 검사한다.
		2.1.2. yes : 검색 주소 배열에 해당 주소정보를 추가한다.
3. 검색 주소 배열을 반환한다.
```

## 주소록 정보를파일에 기록한다

### 함수명

saveAddressBook()

### 정의

주소록정보를 파일에 저장한다

### 반환

X

### 입력

addressBook : personalInfo(10) : 주소록

### 처리과정

```
1. 주소록이 비어있는지 검사한다
	1.1. 빔: 오류 메시지를 출력하고 주소록을 반환한다.
2. 기록할 파일을 open 한다
3. 주소록을 순회한다
	3.1. 주소정보의 삭제여부 속정이 False 인지 확인한다
		3.1.1. yes : 파일에 이름, 전화번호, 주소를 기록한다
4. 파일을 close 한다
```

## 주소록 정보를파일을 불러온다

### 함수명

loadAddressBook()

### 정의

주소록파일을 불러온다

### 반환

addressBook : personalInfo(10) : 주소록

### 입력

addressBook : personalInfo(10) : 주소록

### 처리과정

```
1. 주소록을 모두 비운다
2. 파일을 open 한다
3. 파일 open 이 실패하였는지 검사한다
	3.1. 실패하였으면 오류메시지를 출력한다.
4. 파일 끝까지 반복한다
	4.1. 이름, 전화번호, 주소를 읽어 주소정보에 기록한다
	4.2. 주소정보리스트에 기록한다
5. 파일을 close 한다
```
